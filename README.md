### ☣️ TETE-VIRUS-PLASCOY (Security Lab Simulator)
### ⚠️ AVISO IMPORTANTE

Este projeto é um simulador educacional de comportamento de malware, criado exclusivamente para estudos de cibersegurança em ambientes controlados.

❗ É um vírus real.
❗ DICA NÃO executaR em máquinas pessoais ou sistemas de terceiros.
❗ Uso indevido pode violar leis locais.

### 🎯 Objetivo do Projeto

O TETE-VIRUS-PLASCOY foi desenvolvido para fins educacionais com foco em:

🧪 Simulação de comportamento malicioso
🛡️ Testes de detecção de antivírus (AV/EDR)
📊 Análise de resposta do sistema
🔍 Estudo de engenharia reversa
🧠 Aprendizado em segurança ofensiva e defensiva
🧬 O que este projeto simula

Este laboratório pode simular comportamentos como:

Execução em segundo plano
Processos persistentes (simulados)
Interferência temporária em entradas (teclado/mouse em ambiente controlado)
Contagem regressiva de execução
Logs de atividade simulados
Execução automatizada de rotinas de teste
### ⚠️ Riscos Educacionais ### 

Mesmo em ambiente de teste, estudos desse tipo podem envolver:

Instabilidade do sistema virtual
Consumo de recursos
Simulação de bloqueio de interface
Falsos positivos em antivírus
Interrupção de processos locais
### 🖥️ Ambiente Recomendado

Execute apenas em:

🖥️ Máquina virtual (VirtualBox / VMware)
🧪 Laboratório isolado
🔌 Sistema sem dados pessoais
🌐 Rede NAT ou offline
📸 Snapshots ativados
